# History of Data Technology

데이터는 흔적이자 발전의 재료이다.\
현재 본업을 Data Source를 DR하는 회사에 있기에 관련하여 학습할 예정이다.

## 데이터 기술의 역사

기원전 18000: 엄대, 나무 막대가니 뿔, 뼈 등의 재료에 홈을 파 거래 내역을 표시하는 일종의 출납장부

기원전 2400: 주판, 인류 최초의 계산기

서기 100: 안티키테라, 세계 최초 기계식 계산기

17세기: 통계학

1865: Business Intelligence(리처드 밀러), 처음 상업적 용도로 데이터 분석을 시도한 사례

1881: 홀러리스 도표 작성 기계(천공 카드 사용, 허먼 홀러리스),자동 계산으로 기존 8년 걸릴 데이터 처리 기간을 3개월로 단축

1928: 자기 테이프(프리츠 플로이머)

1943: 콜로서스, 영국이 독일의 로렌츠 암호를 풀기 위해 최초의 데이터 연산 장치이자 디지털 컴퓨터를 생성

1962: 슈박스(윌리엄 C 더쉬), 비정형 데이터의 첫 전산화

1965: 최초의 단일 데이터센터 건설 결정

1970: 관계형 데이터베이스 체계 제시(에드가 F 코드)

1976: MRP 시스템 기업계 보급

1989: WWW 발명

2005: Web2.0 시대 개막, Hadoop 개발

## ETL

Extract, Transform, Load를 나타내며 조직에서 여러 시스템의 데이터를 단일 DB, 데이터 저장소, 데이터 웨어하우스, 데이터 레이크에 결합하기 위해 일반적으로 허용되는 방법이다.

ETL은 데이터 웨어하우징, 머신러닝, 인공지능, 마케팅 데이터 통합, IoT 데이터 통합, DB 복제, 클라우드 마이그레이션 등에 사용된다.

### ETL Tool

#### Integrate.io

여러 데이터 소스를 쉽게 통합할 수 있는 Cloud 기반 ETL 데이터 통합 플랫폼이다.

이 플랫폼에는 많은 소스와 타겟 간 데이터 파이프라인을 구축할 수 있는 간단하고 직관적인 인터페이스가 존재한다.

이 플랫폼은 모든 데이터 볼륨이나 사용 사례에 맞게 확장성이 뛰어나며 데이터를 웨어하우스, DB, OS, 데이터 저장소에 원할히 집계할 수 있다.

Integrate.io의 주요 이점 중 일부는 아래와 같다.

* 높은 확장성과 보안
* 클라우드 기반 ETL 플랫폼
* 여러 데이터 소스를 쉽게 통합
* 간단하고 직관적인 인터페이스

#### Talend Data Integration

On-Premise와 Cloud 모두에서 데이터 소스와 호환되는 오픈 소스 ETL 데이터 통합 솔루션이다.

이 플랫폼에는 수백 개의 사전 구축된 통합이 포함되어 있다.

Talend는 Open Source Version 외에도 생산성, 설계, 관리, 모니터링, 데이터 거버넌스를 위한 추가 도구와 기능이 포함된 유료 데이터 관리 플랫폼을 제공한다.

Talend의 주요 이점 중 일부는 아래와 같다.

* 오픈 소스 및 유료 버전으로 관리
* 디자인, 생산성, 데이터 거버넌스 등을 위한 도구
* On-Premise 및 Cloud 데이터 소스와 호환 가능
* 다목적 데이터 통합 도구

#### IBM DataStage

Client-Server 설계에 중점을 둔 데이터 통합 도구이다.

소스에서 타겟으로 파일, 아카이브, 비즈니스 앱 등을 추출, 변환, 로드한다.

Data Stage를 사용하여 양질의 데이터를 제공해 비즈니스 분석을 지원한다.

DataStage는 필요한 만큼 새로 고치고 동기화 할 수 있어 안정적이고 유연하다.

이기종 소스를 통합하기 위한 쉬운 통합 및 단일 인터페이스를 제공한다.

하드웨어 활용을 최적화하고 수집 및 통합을 지원하며 데이터 통합을 구축, 배포, 업데이트 및 관리하는 방법을 제공한다.

IBM DataStage의 주요 이점 중 일부는 아래와 같다.

* Client-Server 설계
* 소스에서 타겟으로 데이터 추출, 변환, 로드
* 비즈니스 분석 향상
* 다양한 시스템을 함께 연결

#### Oracle Data Integrator

Oracle 데이터 관리 에코 시스템의 일부이다.

Hyperion Financial Management나 Oracle E-Business Suite와 같은 다른 Oracle Application을 이미 사용하고 있을 때 좋은 선택지가 된다.

Oracle Data Integrator는 On-Premise 및 Cloud 버전을 모두 제공한다.

ODI의 고유한 측면 중 하나는 ETL Workload를 지원하여 많은 사용자에게 도움이 될 수 있다는 것이다.

ODI는 대량 배치 로드 및 서비스 지향 아키텍쳐 데이터 서비스와 같은 광범위한 데이터 통합 요청을 지원한다.

병렬 작업 실행을 지원해 더 빠른 데이터 처리를 달성하는 데 도움이 된다.

Oracle Data Integrator의 주요 이점 중 일부는 아래와 같다.

* Oracle 데이터 관리 에코시스템의 일부
* On-Premise 및 Cloud
* ETL Workload 지원
* 병렬 작업 실행

#### Fivetran

API 업데이트를 관리하는 데 도움이 되며 짧은 시간 만에 DB에서 최신 데이터를 가져올 수 있다.

Redshift, BigQuery, Azure, Snowflake와 같은 데이터 웨어하우스와 데이터 통합을 지원하는 Cloud 기반 ETL 솔루션이다.

90개의 SaaS 소스와 사용자 정의 통합을 추가할 수 있는 데이터 소스의 배열이 주요 셀링 포인트이다.

Fivetran의 주요 이점 중 일부는 아래와 같다.

* 편리한 데이터 관리
* 다양한 도구 플랫폼
* API 업데이트 관리
* Cloud 기반 솔루션

#### Stitch

오픈 소스 ETL 데이터 통합 플랫폼이다.

2018년에 Talend에 인수되었다.

Self Service ETL과 자동화된 파이프라인을 제공한다.

130개 이상의 플랫폼, 서비스, Application에서 Data를 소싱하도록 설계되었다.

이 도구는 데이터 웨어하우스의 모든 정보를 중앙 집중화하며 오픈 소스이기에 개발 팀은 추가 소스 및 기능을 지원하도록 도구를 확장할 수 있다.

Stitch의 주요 이점 중 일부는 아래와 같다.

* 오픈 소스, 유료버전 ETL 플랫폼
* Self Service ETL 및 자동화된 파이프라인
* 130개 이상의 플랫폼, 서비스, Application의 소스 데이터

#### Informatica PowerCenter

메타데이터 기반으로 하는 Informatica PowerCenter는 비즈니스 팀과 IT 팀 간 협업을 개선하는 동시에 데이터 파이프라인을 간소화하는 것이 목표인 도구이다.

JSON, XML, PDF와 같은 고급 데이터 형식을 파싱 가능하다.

변환된 데이터를 자동으로 검증해 정의된 표준에 적용할 수 있다.

PowerCenter는 사전 구축된 변환, 고가용성, 최적화된 성능을 제공한다.

Informatica PowerCenter의 주요 이점 중 일부는 아래와 같다.

* 비즈니스 팀과 IT 팀 간 협업 개선
* 데이터 파이프라인 간소화
* 고급 데이터 형식을 구문 분석
* 고성능 및 호환성

#### SAS Data Management

Cloud, Legacy System, 데이터 레이크와 같은 다양한 소스의 데이터를 연결하도록 설계된 데이터 통합 플랫폼이다.

비즈니스 프로세스에 대한 전체론적 보기를 구축하고 WorkFlow를 최적화 할 수 있다.

이 플랫폼은 유연하여 다양한 컴퓨팅 환경 및 데이터베이스에서 작동할 수 있다.

뛰어난 시각화를 생성하는 데 도움이 되는 타사 데이터 모델링 도구와 통합할 수 있다.

SAS Data Management의 주요 이점 중 일부는 아래와 같다.

* 다양한 소스의 데이터 양식 연결
* 비즈니스 프로세스에 대한 전체적인 관점 구축
* 워크플로 최적화
* 다양한 컴퓨팅 환경에서 작동

#### Pentaho

Hitachi Vantara에서 제공하는 오픈소스 플랫폼이다.

Community Edition과 Enterprise Edition의 상용 라이선스를 구매할 수 있다.

초보자도 사용할 수 있는 사용자 친화적 인터페이스를 제공한다.

플랫폼은 표준화된 형식으로 데이터 캡처, 정리, 저장과 같은 데이터 통합 프로세스를 관리한다.

이 도구는 분석을 위해 최종 사용자와 정보를 공유하고 기계 학습에 도움이 되는 IoT 기술에 대한 데이터 엑세스를 지원한다.

Pentaho의 주요 이점 중 일부는 아래와 같다.

* 오픈 소스 플랫폼
* 무료 Community Edition과 Enterprise Edition
* 초보자를 위한 사용자 친화적 인터페이스
* IoT 기술에 대한 데이터 엑세스 지원

#### AWS Glue

AWS에서 제공하는 완전 관리형 ETL 서비스이다.

빅데이터 및 분석 Workload를 위해 설계되었다.

AWS Glue는 ETL Workload를 더 쉽게 만들고 더 큰 AWS 에코 시스템과 잘 통합할 수 있도록 하기 위한 End to End ETL 제품이다.

이 도구의 고유한 측면 중 하나는 Serverless이다.

Amazon은 자동으로 Server를 프로비저닝하고 Workload 완료 후 종료한다.

AWS Glue 스크립트에 대한 작업 예약 및 테스트와 같은 다양한 기능을 제공한다.

AWS Glue의 주요 이점 중 일부는 아래와 같다.

* 완전 관리형 ETL Service
* 빅데이터 및 분석 Workload용으로 설계
* ETL Workload를 더 쉽게 생성
* Workload를 위해 서버를 자동으로 프로비저닝하고 종료
