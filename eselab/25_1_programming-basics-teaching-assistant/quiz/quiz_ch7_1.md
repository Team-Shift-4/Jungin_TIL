# Quiz\_ch7\_1

다음의 빈칸을 알맞게 채우시오.

* 비트 연산자 중 어떤 값의 1의 보수를 얻기 위해 사용하는 연산자는 (     )이다.
* 변수나 수식에서 원하는 비트만을 선택하거나 변경하기 위한 상수나 변수를 (          )라 한다.
* 변수 a가 있을 때, a 값의 최하위부터 n번째 비트를 알아내는 조건식은 (                            )이다.

***

다음 문장에 대해 참/거짓을 작성하시오.

* for 반복문은 조건식이 거짓이더라도 최소 한 번은 반복문이 실행된다.
* 이동 연산자 >>를 수행할 때 비게 되는 가장 왼쪽 비트는 0으로 채운다.
* do while문에서는 continue 문을 사용할 수 없다.

***

다음 프로그램을 수행한 후 출력된 값을 구하시오.

```c
int i = 1, c, num = 14;
for(c = 0; c < 0; c++)
    i *= 2;
printf("%d\n", (num & i) ? 1 : 0);
```

***

다음 프로그램을 수행한 후 출력된 값을 구하시오.

```c
int a = 22, i, sum = 0;
for(i = 1; i < 32; i *= 2) {
    if((a & i) == 0) {
        sum += i;
    }
}
printf("%d\n", sum);
```

다음의 코드를 실행한 후 5를 입력했을 때의 출력값을 적으시오.

```c
scanf("%d", &line);
for(i = 1; i <= line; i++) {
    for(x = i; x > 0; x--) {
        printf("%d", x);
    }
    for(x = 0; x < i; x++){
        printf("%d", x);
    }
    printf("\n");
}
printf("\n");
```
